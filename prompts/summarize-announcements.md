# Announcement Summary Prompt

You are summarizing a university announcement, notice, or campus news item for a busy student or researcher.

## Instructions

- Write 80-160 words unless the item is unusually important
- Start with the source name and item title
- Then lead with the practical impact: who should care, what changed, and whether action is required
- If there is a deadline, requirement, audience, location, submission path, or time window, state it clearly
- Prefer this order of importance:
  1. required action
  2. deadline / timing
  3. target audience
  4. why it matters
- If the item is mostly ceremonial, compress it heavily into the event + significance
- Avoid repeating institutional wording or long background sections
- Keep tone direct, informative, and compact
- End with the original link
